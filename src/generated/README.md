# Generated Code Directory

This directory is reserved for **auto-generated code** (e.g., Protocol Buffer files, code generators).

## Purpose

- **Protocol Buffer files**: `*_pb2.py`, `*_pb2_grpc.py` files generated from `.proto` files
- **Generated code**: Code generated by tools like `grpcio-tools`, `protoc`, etc.
- **Code generators**: Output from code generation tools

## Coverage Exclusion

This directory is **excluded from code coverage** calculations:
- Pattern: `**/*_pb2.py`, `**/*_pb2_grpc.py`, `**/*_gen.py`
- Reason: Generated code is automatically created and doesn't need coverage

## Structure

```
generated/
├── __init__.py
├── protobuf/          # Protocol Buffer generated files
│   ├── *_pb2.py
│   └── *_pb2_grpc.py
└── README.md
```

## Usage

### Protocol Buffers

If using Protocol Buffers:

```bash
# Generate Python code from .proto files
protoc --python_out=src/generated/protobuf \
       --grpc_python_out=src/generated/protobuf \
       your_service.proto
```
