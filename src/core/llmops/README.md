# MOTADATA - LLMOPS

This module provides **LLMOps utilities** for tracking LLM operations like:

- request latency
- token usage
- estimated cost
- success/failure status

## Where the code is

- Implementation: `src/core/llmops/llmops.py`

## When to use this

Enable LLMOps when you need better production visibility into:

- model cost over time
- slow requests
- error rates
