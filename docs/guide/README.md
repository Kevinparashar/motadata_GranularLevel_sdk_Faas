### Using the Guides

1. **Find Documentation by Topic**
   - Open [Documentation Index](DOCUMENTATION_INDEX.md)
   - Browse by functionality, use case, or component

2. **Quick Code Lookup**
   - Open [Quick Reference](QUICK_REFERENCE.md)
   - Find common code snippets and patterns

3. **Navigate Large Documents**
   - Open [Navigation Helper](NAVIGATION_HELPER.md)
   - Learn search strategies and navigation tips

### Using the Search Tool

```bash
# Search for text in documentation
python docs/guide/search_docs.py search "create agent"

# Find docs by topic
python docs/guide/search_docs.py topic "configuration"

# Find function documentation
python docs/guide/search_docs.py function "create_agent"

# List all documentation files
python docs/guide/search_docs.py list
```
